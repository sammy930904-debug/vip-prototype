游戏规则

✕
一、玩法简介

百家乐（Baccarat）是一种纸牌对抗游戏，由庄家（Banker）和闲家（Player）对决，玩家可押注庄赢 / 闲赢 / 和三种主结果，以及庄对 / 闲对 / 幸运 6等副注。每局使用 8 副共 416 张牌，10/J/Q/K 算 0 点，A 算 1 点，其他牌按点数计算，合计只取个位数（如 13 算 3 点）。

三、闲 家 补 牌 规 则

0 - 2必补一张

6闲 6-7 时补
7不补牌

五、胜负判定

补牌后，比较庄家和闲家最终点数，点数大者获胜。如点数相同，则"和"注区获胜，"庄/闲"本金退还（Push）。"庄对/闲对"基于前 2 张是否同点判断，与胜负无关。

六、投注限制

每桌有最小 / 最大投注限额（限红）显示在桌面。投注额超出限红将被拒绝。倒计时结束前投注有效，倒计时结束后下注区锁定，不接受新投注或撤注。
