## 3. 系统设置体系（栈式打开）

点 M3 大厅 / M5 多桌 的 Header 右上角 **⚙ 设置** icon 进入。

### 3.1 进入路径
```
大厅 / 多桌 → ⚙ → 系统设置抽屉（70% 高，从底部弹出）
              ├── 顶部 3 卡：游戏记录 / 游戏规则 / 好路设置
              └── 主菜单列表：
                  ├── 大厅设置 → 二级页（好路提醒 toggle / 轮播广告 toggle）
                  ├── 筹码设置 → dialog（2 选 1 样式 + 常用筹码 11 档勾选）
                  ├── 视频源设置 → dialog（清晰度 + 线路 + 自动缩放 toggle）
                  ├── 音乐设置 → dialog（一键开关 / BGM / 音效）
                  ├── 语言设置 → 二级页（简体中文 / English / 한국어）
                  ├── 安全设置 → 二级页（修改密码）
                  ├── 切换皮肤 → 占位（toast"敬请期待"）
                  ├── 投注方式 → 二级页（确认式 / 点击式）
                  └── 退出游戏（红色危险态）
```

### 3.2 栈式抽屉行为（重点测试）
- 点子项弹出新弹窗时，**父抽屉保留状态**（不关闭，只是被覆盖）
- 关掉子弹窗后**自动回到父抽屉**
- 点 mask（黑色遮罩）= **一键关全栈**
- 点 ✕ = 只关当前

### 3.3 子弹窗高度统一
游戏记录 / 游戏规则 / 好路设置 / 筹码设置 / 视频源设置 / 音乐设置 / 大厅设置 / 投注方式 / 语言设置 / 安全设置 — **全部 70% 屏幕高，底部抽屉风格 + 滑入动画**。
